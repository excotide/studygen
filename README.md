# StudyGen

StudyGen adalah aplikasi belajar berbasis AI untuk membantu kamu belajar lebih cepat dari materi PDF. Cukup unggah materi, lalu aplikasi akan membuat rangkuman yang mudah dipahami dan kuis latihan secara otomatis.

## Tentang Aplikasi

StudyGen dirancang untuk mahasiswa dan pelajar yang ingin:

- Menghemat waktu saat merangkum materi.
- Menguji pemahaman lewat kuis interaktif.
- Menyimpan progres belajar dalam satu tempat.

## Fitur Utama

- Ubah materi PDF menjadi rangkuman otomatis.
- Buat kuis pilihan ganda dari materi yang sama.
- Atur jumlah soal sesuai kebutuhan belajar.
- Simpan riwayat belajar dan nilai terakhir.
- Antarmuka ringan dan mudah digunakan.

## Cara Pakai

1. Login ke akun kamu.
2. Upload file PDF materi.
3. Tunggu proses rangkuman selesai.
4. Lanjutkan dengan generate kuis.
5. Kerjakan kuis dan cek hasilnya.
